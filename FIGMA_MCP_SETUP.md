# Figma MCP Server Configuration

## Setup Instructions

1. Open Cursor Settings (Cmd+, on Mac)
2. Navigate to "MCP Servers" section
3. Add new server:
   - **Name**: `figma`
   - **URL**: `http://127.0.0.1:3845/mcp`
   - **Type**: HTTP

## Usage

Once configured, you can use the Figma MCP server to fetch design specifications directly from your Figma files.

## Server Status

The server is running at: `http://127.0.0.1:3845/mcp`

Note: Make sure the Figma plugin is running and connected to enable the MCP server.
