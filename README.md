# Jan Cermak - Portfolio

Clean, minimalist portfolio website built with vanilla HTML and CSS.

## Features

- Responsive design
- Clean and fast vanilla HTML/CSS (no frameworks)
- Beautiful typography using Boldonse, Bodoni Moda, and Inter fonts
- Optimized for performance

## Setup

Simply open `index.html` in a web browser.

## Design

Design created in Figma, converted to code using Figma MCP server integration.
